PIGMALION CHALLENGE

## Tech Stack

Programming language: TypeScript ^5.3.3.
Node 20.10.0
NPM 10.2.5
Tools:
  - Prettier 3.1.1
  - Jest ^29.7.0

## Run Locally

Clone the project

```bash
  git clone https://github.com/andresfernandez89/challenge-pigmalion.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

Start the test

```bash
  npm run test
```

Measure memory usage

```bash
  npm run performance-memoryUsage
```

Measure execution time

```bash
  npm run performance-executionTime
```
